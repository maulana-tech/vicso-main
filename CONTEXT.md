⚡ What Should I Build?
Build agentic finance applications with SoSoValue's APIs and other supporting tools.

We are looking for agentic challengers to traditional incumbent finance businesses. Leveraging AI and SoSoValue's infrastructure, even a single-person team can build impactful applications that function as a financial news agency, an index publisher or a fund manager --- and make such services available on-chain to users worldwide.

A strong submission should demonstrate:

Clear user value
A clear use case
A complete flow from data input to actionable output
✅ Required
Must genuinely integrate SoSoValue API
Must have a clear use case and real user value
Must complete a basic flow from data input to output
Must provide verifiable demo materials and documentation
⭐ Bonus
Integration with SoDEX API
AI-enhanced functionality
Ability to help users discover opportunities, generate signals, or explain markets
Risk control, confirmation mechanisms, and security awareness
A more complete flow from insight to action
Better product experience (panels, bots, skills, or automated workflows)

For SoSoValue API
Please register for SoSoValue first.
Then register your API access via the SoSoValue API Documentation.
The Buildathon application form above can be used to request higher access limits for your project.

For SoDEX API (Mainnet)
Standard SoDEX mainnet API access is generally available only to users who have reached Silver rank in SoPoints.
By applying through the Buildathon form above, you can request Buildathon whitelist access instead.
Please still register for SoDEX first.
Please note that generating a mainnet API key requires depositing a supported token. Be sure to check the supported chains and the minimum deposit amount in advance.

For SoDEX API (Testnet)
SoDEX testnet can be used directly and does not require a separate API access application.

We usually process applications within one week, so please check the status on your side. If you have any questions or run into any issues, please ask in the Discord Buildathon Channel.

⚖️ Judging Criteria
Category	Weight	Focus
User Value & Practical Impact	30%	Does the project provide clear real-world value and help users gain insight, make decisions, or improve execution efficiency?
Functionality & Working Demo	25%	Is there a clear, functional demo that demonstrates the core features and workflow?
Logic, Workflow & Product Design	20%	Is the product logic clear, well-structured, and supported by a solid workflow or analytical framework?
Data / API Integration	15%	How effectively are SoSoValue API, SoDEX API, and other data sources integrated into the project?
UX & Clarity	10%	Is the product intuitive, well-structured, and easy for users to understand and use?
🧑‍⚖️ Judges
-SoSoValue - A judging team focused on structured financial news, data, and market intelligence.
-SoSoValueIndexes - A judging team focused on on-chain spot indexes, index methodology, and portfolio exposure.
-SoDEX - A judging team focused on on-chain orderbook trading, execution infrastructure, and agent-friendly applications.
